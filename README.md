## Fonctionnalites

- **Theme Dark/Light** : Basculement sombre/clair avec persistance localStorage
- **Trilingue** : French, English, Deutsch avec切换 de langue instantane
- **Animations** : Framer Motion pour transitions, compteurs, cursor custom, effets de typage
- **Cursor personnalise** : Point + anneau suitant la souris (desktop)
- **Fond anime** : Icones tech flottantes avec effet de repulsion souris
- **Scroll progress** : Barre de progression dans la navbar
- **CV Viewer** : Modal pour预visualiser le CV en PDF
- **Responsive** : Mobile-first avec menu hamburger
- **Code splitting** : Contact lazy-load avec React.lazy()

## Installation et utilisation

```bash
# Cloner le depot
git clone https://github.com/tahahb02/portfolio-taha.git

# Installer les dependances
npm install

# Lancer le serveur de dev
npm run dev

# Build pour production
npm run build
```
